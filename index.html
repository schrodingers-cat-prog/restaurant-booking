🚀 Deploy Effortlessly to Vercel

Since this project is hosted on GitHub, you can import it into Vercel within 1 minute with zero configuration changes.

1. Push / Synchronize to GitHub

Ensure your latest changes including this root index.html are pushed to your import repository.

2. Import in Vercel

Log in to your Vercel Dashboard, click "Add New" ➔ "Project", and import your GitHub repository.

3. Root Configuration (Solves 404 Error)

  • Vercel automatically detects the root index.html.
  • Ensure "Framework Preset" is set to "Other" (default).
  • Ensure Root Directory is set to ./ (which is default).
  • Do not change build commands or output directories.
💡 What caused the old 404 error? The repository was previously a pure Android project without any web page in the root. By deploying this new branch which contains index.html in the project root, Vercel will immediately deploy and render this gorgeous interactive website!
